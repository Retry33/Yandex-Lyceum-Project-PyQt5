Пояснительная записка

Описание программы:

	Программа реализует приложение для управления задачами и проектами в стиле системы GTD (Getting Things Done). Она предоставляет 	пользователю интерфейс для добавления, просмотра и управления задачами и проектами.

Основные функции:

1. Главное окно (MainWidget):

	Отображение списка задач и проектов.
	Возможность добавления новых задач с указанием названия, даты и времени.
	Возможность просмотра задач, относящихся к текущему дню.
	Переключение между светлой и темной темой оформления интерфейса.
	Отображение мотивационных сообщений в статусной строке.
	Возможность просмотра и добавления проектов.

2. Окно проектов (ProjectWidget):

	Возможность создания новых проектов с указанием названия и описания.

3. Взаимодействие с базой данных:

	Используется SQLite для хранения информации о задачах (inbox) и проектах (projects).
	Для каждой задачи хранятся название, дата и время.
	Для каждого проекта хранятся название и описание.

4. Внешние зависимости:

	PyQt5/PySide2 для создания графического интерфейса.
	qdarkstyle для добавления стилей темного интерфейса.

5. Инструкции по использованию:

	Запустите программу.
	Добавьте новые задачи через главное окно, указав название, дату и время.
	Переключайтесь между светлой и темной темой оформления по желанию.
	Просматривайте и управляйте проектами через главное окно.
	В окне проектов создавайте новые проекты, указывая название и описание.

Примечания:

	Программа использует базу данных gtd_bd.db (SQLite).
	Для стабильной работы программы убедитесь, что все необходимые библиотеки установлены в вашем окружении Python.
